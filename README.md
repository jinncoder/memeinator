# The Memeinator!

Take memedome to the next level by forcing the desktop background of your victims to the meme of your choosing.

## Quick Start

* Download a release
* Extract the release
* Create a virtual environment
* Install memeinator

```bash
python3 -mvenv ./venv
source ./venv/bin/activate
python3 -mpip install ./meminator*whl
memeinator-download
memeinator
```
